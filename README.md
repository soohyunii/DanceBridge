# DanceBridge

춤을 시작할 용기와, 댄서가 발견될 기회를 연결하는 양면 마켓플레이스.

DanceBridge는 춤을 배우고 싶은 수강생과, 자신에게 맞는 수강생을 만나고 싶은 댄서를 연결하는 서비스입니다. 기존 댄스 플랫폼이 지역·가격·장르 같은 기능적 정보에 머무는 것과 달리, DanceBridge는 "내가 이 수업에 가도 괜찮을까"라는 수강생의 심리적 불안과, "내 춤을 좋아할 사람에게 발견되고 싶다"는 댄서의 니즈에 답합니다.

- [Software Design Document (SDD)](DanceBridge_SDD_v1.txt) — MVP 구현을 위한 시스템 구조, 도메인, 데이터 모델, API, 화면 흐름 가이드

## 핵심 컨셉

- **수강생**: 클래스의 실제 난이도, 분위기, 강사 스타일, 수강생 구성 등 심리적 진입 장벽을 확인하고 자신에게 맞는 수업을 찾습니다.
- **댄서**: 장르, 안무 스타일, 무드, 지도 방식을 구조화한 Dance Profile을 통해 자신과 잘 맞는 수강생에게 발견됩니다.
- **탐색 방식**: 노래 검색, 장르 검색, 무드 검색을 통해 장르를 몰라도 직관적으로 원하는 수업과 댄서를 찾을 수 있습니다.
